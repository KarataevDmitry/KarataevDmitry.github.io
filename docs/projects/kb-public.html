kb-public

Public KB slice for agent memory tooling

All RoslynMcp dotnet-debug-mcp dotnet-build-test-mcp cascade-ide kb-public

kb-public is the published, read-only extract of the agent knowledge layer: knowledge/ playbooks and reference cards, a trimmed hot-context fragment, and routing indexes — without private work/ or personal/ trees from the full canon.

What you get Why it matters
index-knowledge-router-v1.md Fast domain routing under tight context budgets
Integrity kernel (META/integrity-core.md, POST spec) Non-negotiable safety baseline for consumers
Domain playbooks (worlds/*, HCI, Git, KE, …) Operational contracts agents can cite
SHOWCASE.md Onboarding without loading the entire tree

Content is CC BY-SA 4.0 (see knowledge/README.md and PUBLISHING.md in the repo). The MCP server that reads a full private canon is separate: agent-notes-mcp (MIT code).
